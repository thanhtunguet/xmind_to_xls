# XMind to Excel
_Python script to convert Xmind file to Excel (xls)_

## Getting started

- Create virtual environment

    ```zsh
    python3 -m venv .venv
    source .venv/bin/activate
    ```

- Install dependencies

    ```zsh
    pip install -r requirements.txt
    ```

- Convert file

    ```zsh
    python main.py input_file.xmind output_file.xls
    ```
